Offline MD5 Calcurator WebPage.

The server-less web application for calculating MD5 digest for the given files.
It uses:
  • html5 (FILE API)
  • Cyboze Labs' MD5 library to accomplish the job.
[note] Due to the FILE API limitation, it may not work for large files.
